Calculadora Java – Operações Aritméticas

Uma calculadora simples feita em Java que permite ao usuário inserir dois números e escolher uma operação matemática para executar. O sistema funciona via terminal e utiliza a estrutura `switch` para selecionar a operação desejada.

---

Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Módulo (resto da divisão)
- Interface simples no console

---

Tecnologias utilizadas
- Java
- Scanner
- Estrutura de decisão `switch`
